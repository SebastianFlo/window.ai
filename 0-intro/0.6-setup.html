Setup

  1. Download Chrome Canary or Chrome Beta
  2. Go to chrome://flags
  3. Enable "Prompt API for Gemini Nano".
  4. Enable "Enables optimization guide on device"
    1. For some features you will need to also disable "chrome://flags/#text-safety-classifier"
  5. Go to chrome://components
  6. Find "Optimization Guide On Device Model".
  7. Click "Check for Update" and it will install the model locally.
  8. Restart Chrome.
